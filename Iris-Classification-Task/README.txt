# Iris Classification - Internship Task 1

This project is part of my Data Science internship.  
Objective: Classify iris flowers into species using ML models.

## Steps Done
- Data exploration & visualization
- Preprocessing
- Model training (Logistic Regression, KNN, Decision Tree, Random Forest, SVM)
- Hyperparameter tuning
- Model comparison & conclusion

## Results
- Best Model: Random Forest
- Tuned Accuracy: 99%

## Requirements
- Python 3.x
- pandas, numpy, matplotlib, seaborn, scikit-learn
